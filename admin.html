FormSubmit.co Setup
How to activate email notifications when a contact form is submitted.
1

Activate FormSubmit

Open your index.html and confirm the form action is set to:

action="https://formsubmit.co/amtechiesprinter@gmail.com"

2

First Submission Activates It

When the first form is submitted, FormSubmit.co will send a confirmation email to amtechiesprinter@gmail.com. Click the confirmation link to activate.

3

All Future Submissions Go to Gmail

After activation, every support form submission will be emailed to amtechiesprinter@gmail.com with full customer details.

How to Find Your Tawk.to IDs
Follow these steps to get your unique widget embed code.
1

Log in to Tawk.to

Go to https://tawk.to and sign in with amtechiesprinter@gmail.com

2

Navigate to Chat Widget

Go to: Administration → Channels → Chat Widget

3

Click the "JavaScript" Tab

You will see embed code like:

s1.src='https://embed.tawk.to/PROPERTY_ID/WIDGET_ID';

4

Copy Both IDs Here

Paste the Property ID and Widget ID into the fields above and click Save.

5

Update index.html

Use the generated embed code preview to update the tawk.to script section in your index.html.

Google Site Verification
Enter your Google verification code for Google Ads / Search Console. This goes in the <head> meta tag.
Add this to your index.html inside <head>:
<meta name="google-site-verification" content="YOUR_CODE_HERE" />
How to get Google Verification Code
Steps to verify your site with Google Search Console (required for Google Ads).
1

Go to Google Search Console

Visit search.google.com/search-console

2

Add a Property

Click "Add Property" and enter your website URL (e.g. https://amtechiesprinter.com)

3

Choose HTML Tag Method

Select "HTML tag" as the verification method. You'll get a meta tag code.

4

Copy the content= value

Copy just the content value (not the full tag) and paste it in the field above.
